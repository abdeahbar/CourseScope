# Introduction to Data Analysis with Python

## Course description

This course introduces students to practical data analysis using Python.
Students will learn how to load datasets, clean data, explore patterns, and
create simple visualizations.

## Prerequisites

- Basic Python syntax, including variables, functions, and lists
- Basic understanding of tables and spreadsheets

## Learning objectives

By the end of this course, students will be able to:

- Read CSV files into Python
- Clean missing and inconsistent values
- Summarize data using simple statistics
- Create charts to communicate findings

## Expected results

After studying this course, students should be able to complete a small data
analysis project from raw data to a short written summary.

## Topics

1. Working with files
2. Data cleaning
3. Exploratory data analysis
4. Basic visualization
5. Communicating results

## Assessment

Students will analyze a small dataset and explain their main findings in a
short report.
